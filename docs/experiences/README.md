## 如何使用

1. **搜索**：按关键词或时间查找相关经验
2. **复用**：将解决方案应用到类似场景
3. **引用**：在 commit 或文档中引用对应的经验文件链接

## 经验文件模板

每个经验文件应包含：
- 问题场景
- 根本原因
- 解决方案
- 验证方式
- 适用场景
- 相关文件链接

参考示例：`2026-03-25-frontend-visual-design-confirmation.md`
